2.3            Multidimensional

1.  The directory containing files of this external library is: installation directory\esProc\extlib\OlapCli. The Raqsoft core jar for this external library is scu-olap4j-cli-2.10.jar.

olap4j-1.2.0.jar

olap4j-xmla-1.2.0.jar

Note: The third-party jars are provided within the package and users can choose appropriate ones for specific scenarios.

 

2.  Olap4jCli requires a JRE version 1.7 or above. Users need to install a higher version if the esProc built-in JRE version does not meet the requirements, and then configure java_home in config.txt under installation directory \esProc\bin. Just ignore this step if the JRE version is adequate.

 

3.  esProc provides three external library functions - olap_open(), olap_query() and olap_close() - to access multidimensional databases. Look them up in【Help】-【Function reference】to find the uses.
