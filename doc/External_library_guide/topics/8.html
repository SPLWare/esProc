2.6            Financial

1.The directory containing files of this external library is: installation directory\esProc\extlib\FinancialCli. The Raqsoft core jar for this external library is scu-financial-cli-2.10.jar.

2. esProc provides a series of external library functions – Faccrint(), Faccrintm(), Fcoupcd(), etc. – to access the Financial external library. Look them up in【Help】-【Function reference】to find their uses.
