msum()

Description:

Calculate sum on a matrix or a multidimensional matrix.

Syntax:

msum(A, n)

Note:

An external library function where parameter n is the ordinal number of level on which the summarization is performed. If n is absent, the desired dimension is the first one whose size is not 1 in matrix A.

Parameter:

A

n

A matrix or a multidimensional matrix

A positive integer

Option:

@a

Calculate sum on all elements of matrix A and parameter n is invalid

Return value:

A sequence or a number

Example:
