ali_close()

Description:

Close connection to Alibaba Cloud.

Syntax:

ali_close(ali_open)

Note:

This external library function (See External Library Guide) closes the connection to Alibaba Cloud.

Parameter:

ali_open

A connection to Alibaba Cloud

Example:

 

A

 

1

=ali_open("http://test.ots.aliyuncs.com","LTAIXZNG5zzSPHTQ","YHJHhdf","test")

 

2

=ali_close(A1)

Close connection to Albaba Cloud.
