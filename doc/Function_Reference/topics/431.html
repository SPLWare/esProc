ifa()

Description:

Find whether an object is a sequence.

Syntax:

ifa(x)

Note:

The function finds whether the parameter x is a sequence.

Parameter:

x

The object to be judged

Return value:

A boolean value

Example:

 

A

 

1

=ifa([1,2,3])

True

2

=ifa(123)

False
