redis_zsize()

Description:

Get the number of records in an ordered set.

Syntax:

redis_zsize(con,key)

Note:

The external library gets the length, that is, the number of records in an ordered set.

Parameters:

con

Database connection string

key

Name of the ordered set

Return value:

An integer

Example:

 

A

1

=redis_open()

2

= redis_zsize(A1, "runoobkey")
