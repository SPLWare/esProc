T.groupn(x;F)

Description:

Group records of a pseudo table and write the grouped subsets to a sequence of bin files.

Syntax:

T.groupn(x;F)

Note:

The function groups records of pseudo table T by grouping expression x and writes the grouped subsets to a sequence of bin files F.

Parameter:

T

A pseudo table

x

Grouping expression

F

A sequence of bin file objects

Return value:

Pseudo table:

Example:
