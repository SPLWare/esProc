abs()

Description:

Compute absolute value of the specified data.

Syntax:

abs(numberExp)

Note:

The function computes the absolute value of numberExp.

Parameter:

numberExp

Data for which you want to compute the absolute value

Return value:

Numeric

Example:

abs(-3245.54)

3245.54

abs(-987)

987
