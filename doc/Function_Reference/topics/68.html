blob()

Description:

Convert a blob object into an integer sequence, or vice versa.

Syntax:

blob(b )

Note:

The function converts the blob object into a data column or an integer sequence into a blob object. Each byte of blob object corresponds to a member of the would-be integer sequence.

Parameters:

b

A blob object or an integer sequence

Return value:

A blob object or an integer sequence

Example:

 

A

 

1

=blob([4,7,3])

Return a blob object

2

=blob(A1)

Return an integer sequence
