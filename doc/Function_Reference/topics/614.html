redis_hscan()

Description:

Scan the key value pairs iteratively in a HASH table.

Syntax:

redis_hkeys(con,tbl,regex,[count])

Note:

The external library function scans the key value pairs iteratively in a HASH table and stores them in an already defined data structure. Error will be reported if the structure isn¡¯t defined in advance.

Parameters:

con

Database connection string

tbl

HASH table name

regex

A regular expression

count

Number of key values scanned at a time

Return value:

A table sequence
