redis_hsize()

Description:

Get the number of fields in a HASH table.

Syntax:

redis_hsize(con,tbl)

Note:

The external library function returns 0 when key is absent.

Parameters:

con

Database connecton string

tbl

HASH table name

Return value:

A string

Example:

 

A

 

1

=redis_open()

 

2

=redis_hsize(A1,"da")

3

=redis_hsize(A1,"db")
