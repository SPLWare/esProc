hash()

Description:

Return a hash value.

Syntax:

hash(xi,…;n)

Note:

The function calculates the hash value(s) over data items xi,… and returns a positive integer less than parameter n.

Parameter:

xi

Data of any type

n

An integer greater than 1

Return value:

A positive integer

Example:

=hash(0.2;38)

8

=hash(2,52,94;100)

29

=hash("abc";10477)

2063
