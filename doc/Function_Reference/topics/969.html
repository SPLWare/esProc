tinv()

Description:

T inverse cumulative distribution function.

Syntax:

tinv(p, nu)

Note:

External library function (See External Library Guide), which is Tinverse cumulative distribution function.

Parameter:

P

Probability within the interval (0,1) An integer

nu

Degrees of freedom corresponding to probabilities An integer

Return value:

Double-precision number

Example:

 

A

 

1

=tinv(0.99, 2)
