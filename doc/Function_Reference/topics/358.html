r.fname(i)

Description:

Get field names from records according to sequence numbers of these fields.

Syntax:

r. fname(i)

Note:

The function gets the name of the field whose sequence number is i from record r. If no parameter is present, then return all field names in r.

Parameter:

r

Record

i

Sequence number of a field

Return value:

Character string/Sequence

Example:
