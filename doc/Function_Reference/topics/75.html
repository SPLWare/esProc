acosh()

Description:

Return the inverse hyperbolic cosine.

Syntax:

 acosh(number)

Note:

 The function computes the inverse hyperbolic cosine of a real number.

Parameter:

number

A real number.

Return value:

Numeric value

Example:

acosh(10)

2.993222846126381
