arguments(spl)

Description:

Return a list of the specified script files argument names.

Syntax:

arguments(spl)

Note:

The function returns a list of the specified .dfx/.splx/.spl script files argument names. It returns argument names in the current script file when the parameter is absent.

Parameter:

spl

Name of a script file in the format of .dfx/.splx/.spl

Return value:

A sequence

Example:

Set cellset arguments for test.splx:

Set cellset arguments for the current script file:

 

 

 

A

 

1

=arg1+" world!"

Result: Hello world!

2

=arg2+88

Result: 108

3

=arguments()

Return a list of the current script files argument names

4

=arguments("test.splx")

Return a list of argument names from script file test.splx
