sleep()

Description:

Make esProc program sleep.

Syntax:

sleep(n)

Note:

The function makes esProc program sleep for n milliseconds.

Parameter:

n

Millisecond

Example:

sleep(10000)

Suspend esProc program for 10000 milliseconds before resuming execution
