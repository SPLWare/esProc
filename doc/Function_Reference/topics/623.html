redis_open()

Description:

Connect to redis database.

Syntax:

redis_open(xmlFile)

Note:

The external library function connects to the redis database. The configuration file to be parsed only supports XML format. The XML file is located in [installation directory]/extlib/RedisCil. When no option is present, the connection style is the single-instance mode and the corresponding configuration file is spring-redis.xml.

Parameters:

xmlFile 

The XML file to be parsed; load corresponding configuration file according to the present function option when the parameter is absent

Options:

@c

Cluster style conncetion using configuration file spring-cluster.xml

@s

Sentinel style conncetion using configuration file spring- sentinel.xml

Return value:

A cursor
