memory(h,V)

Description:

Generate a cluster in-memory table from a local in-memory table.

Syntax:

memory(h,V)

Note:

The function generates a cluster in-memory table from local in-memory table V, which is stored in a global variable, held in node sequence h. Only supported by esProc Enterprise Edition.

Parameter:

h

A node sequence.

V

An in-memory table’s global variable.

Return value:

Cluster in-memory table

Example:

Suppose the global variable arg1 is already assigned value using env(v, data) in the initiation script init.splx:

 

A

 

1

=memory(["192.168.31.72:8281","192.168.31.72:8291"],arg1)

Generate a cluster in-memory table from the local in-memory table arg1.
