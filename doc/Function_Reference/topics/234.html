f.date()

Description:

Return the time and date of a file last modified.

Syntax:

f.date()

Note:

The function returns the time and date of file f last modified.

Parameter:

f

File object.

Return value:

Datetime data

Example:

 

A

 

1

=file("D://p1.splx").date()

"2013-05-02 09:45:08"
