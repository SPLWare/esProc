A.icount()

Description:

Count the number of distinct members in a sequence.

Syntax:

A.icount()

 

Note:

The function calculates the number of distinct members in a sequence A.

Parameters:

A

A sequence of n members

Options:

   @o  Remove duplicates by comparing neighboring members

Return value:

A numerical value

Example:

 

A

 

1

=["a","c","d","e","f","a","a"]

 

2

=A1.icount()

5

3

=A1.icount@o()

6
