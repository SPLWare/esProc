ym_server_quit()

Description:

Close Python service.

Syntax:

ym_server_quit()

ym_server_quit(1)

Note:

The external library function closes Python service.

Example:

 

A

 

1

=ym_env()

Initialize the environment to start Python service

2

>ym_close(A1)

Release resources

3

>ym_server_quit()

Close Python service
