sinh()

Description:

Return the hyperbolic sine.

Syntax:

sinh(number)

Note:

The parameter number is any real number.

Parameters:

number

The real number for which you want to find the hyperbolic sine

Return value:

The hyperbolic sine

Example:

sinh(1)

1.1752011936438014
