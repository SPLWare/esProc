dql()

Description:

Create DQL connection.

Syntax:

dql(meta)

Note:

The function creates a DQL connection; support both absolute path and relative path.

Parameter:

meta

The metadata file

Example:

 

A

 

1

=dql("demo.glmd")

Create DQL connection and set the metadata file as demo.glmd in the main directory.
