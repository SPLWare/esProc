acos()

Description:

Compute arc cosine value of the specified data.

Syntax:

acos(number)

Note:

The parameter number is a real number between -1 and 1.

Parameter:

number

The real number for which you want to compute the arc cosine

Return value:

Arc cosine

Example:

acos(-1)

3.141592653589793

acos(cos(pi()/2)) 

1.5707963267948966

acos(cos(0))

0.0

asin()
