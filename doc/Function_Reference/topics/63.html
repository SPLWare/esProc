acosh()

Description:

Return the inverse hyperbolic cosine.

Syntax:

 acosh(number)

Note:

 The parameter number is a real number.

Parameters:

number

The real number for which you want to find the inverse hyperbolic cosine

Return value:

The inverse hyperbolic cosine

Example:

acosh(10)

2.993222846126381
