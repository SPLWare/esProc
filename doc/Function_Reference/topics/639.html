redis_sscan()

Description:

Scan elements of the key in a set iteratively.

Syntax:

redis_hkeys(con,key,regex,[count])

Note:

The external library scans elements of the key in a set iteratively and stores the data in a corresponding data structure. Error will be reported if such a data structure has not defined at store.

Parameters:

con

Database connection string

key

Set name

regex

A regular expression

count

Number of elements in a key scanned at a time

Return value:

A table sequence

Example:

 

A

 

1

=redis_open()

 

2

=redis_sscan(A1, "runoobkey", "*i*", 3)
