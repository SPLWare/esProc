ifpure()

Description:

Judge whether a data object is a pure sequence or a pure table sequence.

Syntax:

ifpure(x)

Note:

The function judges whether data object x is a pure sequence or a pure table sequence.

Parameter:

x

Any data object, including constant, expression, record, sequence, table sequence and record sequence

Return value:

Boolean

Example:

Judge whether a data object is pure sequence:

 

A

 

1

=["a","w","s"]

A sequence.

2

=ifpure(A1)

Judge whether A1 is a pure sequence or a pure table sequence, and return false.

3

=A1.i()

Convert A1 into a pure sequence.

4

=ifpure(A3)

Return true.

Judge whether a data object is pure table sequence:

 

A

 

1

=3.new(~:ID,~*~:Num)

A table sequence.

2

=ifpure(A1)

Judge whether A1 is a pure sequence or a pure table sequence, and return false.

3

=A1.i()

Convert A1 into a pure table sequence.

4

=ifpure(A3)

Return true.
