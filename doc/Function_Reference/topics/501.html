mse()

Description:

Calculate the standard error of the mean between two vectors.

Syntax:

mse(A,B)

Note:

The function calculates the standard error of the mean between vector A and vector B.

Parameters:

A/B

Vectors

Return value:

Numeric value

Example:

 

A

 

1

[[3,5],[2,8]]

 

2

[[4,6],[2,4]]

 

3

=mse(A1,A2)

4.5

4

=mse([1],[8])
