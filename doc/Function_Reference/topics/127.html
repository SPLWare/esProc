A.count()

Description:

Count the number of non-null members in a sequence.

Syntax:

A.count()

Note:

The function counts the number of the non-null members in the sequence A.

Parameters:

A

A sequence

Return value:

An integer

Example:

 

A

 

1

=[1,2,3,4].count()

4

2

=[1,null,3,4].count()

3

3

=count(1,null,3,4)

3
