upper()

Description:

Convert all characters in a string to upper case.

Syntax:

upper(s)

Note:

The function converts all characters in a string to upper case.

Parameter:

s

Source string to be converted to upper case

Option:

@q

The quoted strings won’t be converted

Return value:

String

Example:

upper("ABCdef")

"ABCDEF"

upper("abcDEF")

"ABCDEF"

upper@q("\'abCD\'")

'abCD'

 

 

A

 

1

"ABcde"

 

2

=upper@q(A1)

"ABcde"
