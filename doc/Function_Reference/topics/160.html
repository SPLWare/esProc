qvm.close()

Description:

Close QVM.

Syntax:

qvm.close()

Note:

The function closes QVM.

Parameter:

qvm

The QVM object.

Example:

 

A

 

1

=Qconnect("http://54.213.221.123:8080/qvs":"aws-qvs",30,300)

Request QVA through QVS to access QVM and return a QVM object.

2

=A1.exec("qtc01/imp_txt.splx";100)

Execute SPL script file.

3

>A1.close()

Close QVM.
