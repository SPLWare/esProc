iftime()

Description:

Find whether the specified data is time type.

Syntax:

iftime(exp)

Note:

The function finds whether parameter exp is of time type.

Parameter:

exp

An expression of any data type

Return value:

Boolean

Example:

iftime("10:20:30")

false

iftime(time("10:20:30"))

true

iftime("2006-10-10")

false

iftime("2006-10-10 10:20:30")

false

iftime("20061010")

false

Related functions:

ifdate()
