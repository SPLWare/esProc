stax_cursor()

Descripiton:

Query the Cassandra database and return result as a cursor.

Syntax:

stax_cursor(staxClient, cql, [arg1], [arg2], ...) 

Note:

The CassandraCli external library function (See External Library Guide) queries a Cassandra database and returns the result value as a cursor.

Parameter:

staxClient

The database connection handle.

cql

A CQL query statement.

args

A parameter value; can be absent.

Example:

 

A

 

1

=stax_open("127.0.0.1":9042, "mycasdb","cassandra":"cassandra")

Cassandra database connection.

2

=stax_cursor(A1,"select * from user")

Perform the query and return result as a cursor.

3

=A2.fetch()

 

4

=stax_close(A1)
