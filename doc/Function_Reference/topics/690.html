A.range()

Description:

Adjust members of a numeric sequence to a specified interval.

Syntax:

A.range(min, max)

Note:

The external library function adjusts members of numeric sequence A to a specified interval [min, max].

Parameters:

A

A numeric sequence

Return value:

A sequence

Example:

 

A

 

1

[0,1,2,3,4,5,6,100]

 

2

=A1.range(1,10)
