date(datetimeExp)

Description:

Get the date part of the datetime value.

Syntax:  

date(datetimeExp)

Note:  

The function gets the date part of the datetimeExp.

Parameter:  

datetimeExp

Datetime value

Return value:  

Date type data

Example:

date(now())

2013-12-09

Related function:

date()
