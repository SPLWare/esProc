comsign()

Description:

A sign function that only handles complex number.

Syntax:

A.comsign()

Note:

The external library function returns a same-size sequence where each element is A/A.comabs().

Parameter:

A

A sequence of complex numbers

Return value:

A sequence

Example:

 

A

 

1

=[[1,2],[3,4],[5,6],[7,8],[5,0],[-5,0],[-8,-9],[-0.3,5],[3,-4],[0,0],[0,3],[0,-2]]

 

2

=A1.comsign()
