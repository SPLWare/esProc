clear

Description:

Clear the cell value.

Syntax:

clear a:b,c,d:e

Note:

The function clears the values of certain cells. With only a: specified, it clears the cell values in a code block where a is the master cell.

Parameters:

a:b,c,d:e

The cells whose values need to be cleared.
