mmean()

Description:

Calculate the mean value within a matrix or a multidimensional matrix.

Syntax:

mmean(A, n)

Note:

An external library function where parameter n is the sequence number of level on which the summarization is performed. If n is absent, the desired dimension is the first one whose size is not 1 in matrix A.

Parameter:

A

n

A matrix or a multidimensional matrix

A positive integer

Option:

@a

Calculate the mean on all elements of matrix A and parameter n is invalid

 Return value:

A sequence or a number
