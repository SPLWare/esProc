file(fn:z)

Description:

Return a file or file group preceded by the corresponding zone table number(s).

Syntax:

file(fn:z)

Note:

The function returns a file preceded by the corresponding zone table number z or a file group where each file is preceded by the corresponding zone table number z from composite table file fn. The format of file name is z.fn. The function returns a file when parameter z is an integer, and a file group when it is an integer sequence.

Parameters:

fn

A composite table file

z

An integer or an integer sequence

Return value:

 Name of a composite table file or a file group object

Example:

 

 

A

 

1

=file("emp1.ctx":1)

Generate composite table file 1.emp1.ctx

2

=A1.create(#EID,NAME,SALARY;1)

 

3

=file("emp.ctx":[2,3,4])

Generate a file group that contains files 2.emp.ctx, 3.emp.ctx and 4.emp.ctx

4

=A3.create(#EID,NAME,SALARY)
