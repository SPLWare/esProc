dynadb.close()

Description:

Close connection to Amazon DynamoDB.

Syntax:

dynadb.close()

Note:

The external library function (See External Library Guide) closes connection to DynamoDB.

Example:

 

A

1

=dyna_open("http://localhost:8000", "us-east-2")

2

>A1.close()
