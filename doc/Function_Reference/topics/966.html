zip_open()

Descripition:

  Open a zip file.

Syntax:
  zip_open(zipfile:encoding,passowrd)

Note:

The external library function (See External Library Guide) opens a zip file.

Parameter:

zipfile

A compressed file or a file object

encoding

Character set, which is UTF-8 by default

passowrd

A password; can be omitted

Return value:
Zip file object

Example:

 

A

 

1

=zip_open("D:/z2.zip")

 

2

=zip_open("D:/z2.zip":"GBK","123")
