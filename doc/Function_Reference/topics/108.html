blob()

Description:

Convert a blob object into an integer sequence, or vice versa.

Syntax:

blob(b )

Note:

The function converts the blob object into a data column or an integer sequence into a blob object. Each byte of blob object corresponds to a member of the would-be integer sequence.

Parameter:

b

A blob object or an integer sequence

Option:

@4

Every 4 bytes correspond to one int

@8

Every 8 bytes correspond to one int

@f

Every 4 bytes correspond to one float

@d

Every 8 bytes correspond to one double

@r

All the above options use little-endian; by default, use big-endian

Return value:

A blob object or an integer sequence

Example:

 

A

 

1

=blob([4,7,3])

Return a blob object

2

=blob(A1)

Return an integer sequence

3

=blob@4([4,7,3])

 

4

=blob@r4(A3)

Return an integer sequence; @r4 options enable using the little-endian and every 4 bytes correspond to one int
