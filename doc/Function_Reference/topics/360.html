for a,b,s

Description:

Execute a loop according to the specified scope and step.

Syntax:

for  a,b,s

Note:

The statement executes the code block in loops from a to b with a step of s.

Parameter:

a

An integer

b

An integer

s

An integer to indicate the step. Its value is 1 by default

Example:
