Modulus

Description:

Divide two long integers/real numbers and get the remainder or an integer result.

 

Syntax:

x%y

Divide x by y and get remainder.

x\y

Retain the integer part of x and y to perform the division and return an integer result.

Parameter:

x

A long integer/real number.

y

A long integer/real number.

Return value:

Integer/Real number

Example:

 

A

 

1

=7%2

1

2

=7\2

3
