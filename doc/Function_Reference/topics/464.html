influx_close()

Description:

Close InfluxDB connection.

Syntax:

influx_close(hd)

Note:

The external library function closes InfluxDB connection. It is valid only with InfluxDB 1.x.

Parameter:

hd

Connection object

Return value:

None

Example:

 

A

 

1

=influx_open("http://127.0.0.1:8086", "mydb", "autogen", "admin", "admin")

 

 

2

=influx_close(A1)
