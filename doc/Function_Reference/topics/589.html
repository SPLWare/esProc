ln()

Description:

Compute the natural logarithm.

Syntax:

ln(numberExp)

Note:

The function computes the natural logarithm of parameter numberExp.

Parameter:

numberExp

A positive number.

Return value:

Numeric value

Example:

ln(54)

3.9889840465642745

Related function:

lg()
