atanh()

Description:

Return the inverse hyperbolic tangent.

Syntax:

atanh(number)

Note:

The parameter is a real number.

Parameter:

number

Any real number between -1 and 1

Return value:

The inverse hyperbolic tangent

Example:

atanh(0.5)

0.5493061443340549
