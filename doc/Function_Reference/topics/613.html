redis_hexists()

Description:

Check whether a specified field exists in a HASH table.

Syntax:

redis_hexists(con,tbl,field)

Note:

The external library function checks whether there is a certain field field in HASH table tbl.

Parameters:

con

Database connection string

tbl

HASH table name

field

A field name

Return value:

Boolean

Example:

 

A

1

=redis_open()

2

=redis_hexists(A1,"empname","Tom")
