det()

Description:

Calculate the determinant of a square matrix.

Syntax:

det(A)

Note:

The function computes the determinant of square matrix A.

Parameter:

A

Square matrix

Return value:

Numeric value

Example:

 

A

 

1

[[0,1,2],[1,0,3],[4,-3,8]]

 

2

=det(A1)

-2.0
