sf_wsdlview()

Description:

Query data in an Sobject object and return a table sequence made up of field names and field types.

Snytax:

sf_wsdlview(sobject)

Note:

The external library function queries data in an Sobject object and returns a table sequence made up of field names (FieldName) and field types (FieldType).

Parameters:

sobject

The Soobject data object to be queried

Return value:
A table sequence

Example:

 

A

 

1

=sf_wsdlopen("user","pwd")

 

2

=sf_wsdlquery(A1, "Account")

3

=sf_wsdlclose(A1)
