T.reset()

Description:

Clear the members of a table sequence.

Syntax:

T.reset()

Note:

The function clears members from table sequence T, but reserves the data structure.

Parameter:

T

A table sequence

Return value:

Empty table sequence

Example:

 

A

 

1

=demo.query("select * from EMPLOYEE")

2

=A1.reset()
