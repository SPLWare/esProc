var()

Description:

Calculate the generalized variance of a vector (sequence).

Syntax:

var(V)

Note:

The function computes the generalized variance of vector V.

Options:

@s

Calculate sample variance of a vector by dividing by n-1

Parameters:

V

A vector

Return value:

Numeric value

Example:

 

A

 

1

[4,8,12,21]

 

2

=var([3,8])

6.25

3

=var(A1)

39.6875

4

=var@s(A1)

52.916666666666664
