sin()

Description:

Compute the sine value.

Syntax:

sin(number)

Note:

The parameter number is in radians.

Parameters:

number

Radians for which you want to compute the sine

Return value:

float type

Example:

sin(pi())

1.2246467991473532E-16

sin(pi(2))

-2.4492935982947064E-16

sin(pi()/2)

1.0

Related functions:

cos()

tan()
