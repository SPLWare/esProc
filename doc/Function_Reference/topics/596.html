rankm()

Description:

Return the rank of a matrix.

Syntax:

rankm(A)

Note:

The function returns the rank of matrix A.

Parameters:

A

A matrix

Return value:

A numeric value

Example:

 

A

 

1

[[1,-2,1],[0,2,-8],[-4,5,9]]

 

2

=rankm(A1)
