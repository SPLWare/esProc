product()

Description:

Return the product of a set of numbers.

Syntax:

product(xi,…)

product(A)

Note:

The function multiplies the members of [x1, x2,…], where any member that is not a number will be ignored.

Parameter:

A

A sequence

xi

An expression

Example:

 

A

 

1

=product(17,3,5)

255.0

2

=product(4,"b",12,2)

96.0
