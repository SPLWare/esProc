cos()

Description:

Compute the cosine value.

Syntax:

cos(numberExp)

Note:

The parameter numberExp is defined in radians.

Parameters:

numberExp

The radian number of the cosine to be computed

Return value:

Float type

Example:

cos(pi())

-1.0

cos(pi(2))

1.0

Related functions:

sin()

tan()
