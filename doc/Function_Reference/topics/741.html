sf_wsdlclose()

Description:

Close the cloud server Salesforce server connected through soap api.

Syntax:

sf_wsdlclose(fd)

Note:

The external library function closes the cloud server Salesforce server connected through the way of soap api to release resources.

Parameters:

fd

restclient object

Example:

 

A

 

1

=sf_wsdlopen("user","pwd")

Connection

2

=sf_wsdlclose(A1)

Close the connection
