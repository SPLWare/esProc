remainder()

Description:

Calculate remainder for degree to radian conversion.

Syntax:

remainder(x,y)

Note:

The function calculates remainder from the division of one real number by another for degree to radian conversion. The range of degrees is 0°-360° and the range of radians is 0-2*pi.

Parameter:

x

An integer or a real number

y

An integer or a real number

Return value:

An integer or a real number

Example:

 

A

 

1

=remainder(-450,360)

270

2

=remainder(-5,90)

85
