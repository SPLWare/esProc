r.fno(F)

Description:

Get the sequence number of a field of a record in a table sequence.

Syntax:

r.fno(F)

Note:

The function gets the sequence number of field F of record r in the table sequence; if there is no parameter, return the number of fields in the table sequence in which r locates. If the field cannot be found, a null value will be returned.

Parameter:

r

A record

F

Field name

Return value:

The sequence number of a field

Example:

 

A

 

1

=demo.query("select * from EMPLOYEE")

 

2

=A1(2).fno(EID)

1, which indicates that the sequence number of the field "EID" is 1.

3

=A1(2).fno()

9, which indicates that there are 9 fields in the table "Employee".
