ym_save_pcf()

Description: 

Generate a model file according to a model object.

Syntax:

ym_save_pcf(pd,pdfile)

Note:

The external library function (See External Library Guide) generates a model file with the extension pcf according to model object pd. The file can be stored in an absolute path or a relative path; the default relative path is [sAppHome]/store/predict.

Parameter:

pd

A model object

pdfile

A model file with the extension pcf

Return value:

Boolean value
