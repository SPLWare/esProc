es_close()

Description:

  Close REST Client connection.

Syntax:

   es_close(fd)

Note:

The external library function closes connection to REST Client and release resources.

Parameter:

fd

A REST Client object

Example:

 

A

 

1

=es_open("localhost:9200")

Connect to ES server

2

=es_close(A1)

Close the connection
