fact()

Description:

Compute factorial of the specified data.

Syntax:

fact(nExp)

Note:

The function computes the factorial of nExp.

Parameter:

nExp

Data for which you want to compute the factorial

Return value:

Long integer (64 bit)

Example:

fact(10)

3628800
