norminv()

Description:

An inverse normal cumulative distribution function.

Syntax:

norminv(p,mu,sigma)

Note:

The external library function (See External Library Guide) is an inverse normal cumulative distribution function.

Parameter:

P

Probability, within the interval (0,1) .

mu

The mean value.

sigma

The standard deviation, which is a positive number.

Return value:

Double-precision number

Example:

 

A

 

1

=norminv(0.25, 2, 1)
