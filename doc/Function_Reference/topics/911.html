tanh()

Description:

Return the hyperbolic tangent.

Syntax:

tanh(number)

Note:

The parameter number is a real number.

Parameter:

number

The real number for which you want to find the hyperbolic tangent

Example:

tanh(0.5)

0.46211715726000974
