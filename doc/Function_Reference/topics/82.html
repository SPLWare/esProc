asinh()

Description:

Return the inverse hyperbolic sine.

Syntax:

asinh(number)

Note:

The parameter number is a real number.

Parameter:

number

The real number for which you want to find the inverse hyperbolic sine

Return value:

The inverse hyperbolic sine

Example:

asinh(10)

2.99822295029797
