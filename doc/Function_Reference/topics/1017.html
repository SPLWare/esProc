ym2_pcfload()

Description:

Read a pcf file as a pcf object.

Syntax:

ym2_pcfload(pcfFile)

Note:

External library function (See YModel User Reference).

 

The function reads a pcf file as a pcf object.

Parameter:

pcfFile

pcf file name.

Return value:

pcf object/false

Example:

 

A

1

=ym2_env("F:/ymodel")

2

=file("F:/ymodel/data/train.pcf")

3

=ym2_pcfload(A2)

4

=ym2_pcfload("F:/ymodel/data/train.pcf")

5

F:/ymodel/data/train.pcf

6

=ym2_pcfload(A5)

7

>ym2_close()
