comimage()

Description:

Get a complex number’s imaginary part.

Syntax:

A.comimage()

Note:

The external library function (See External Library Guide) gets the imaginary part of a complex number.

Parameter:

A

A sequence of complex numbers

Return value:

Sequence

Example:

 

A

 

1

=[[1,2],[3,4],[5,6],[7,8],[5,0],[-5,0],[-8,-9],[-0.3,5],[3,-4],[0,0],[0,3],[0,-2]]

 

2

=A1.comimage()

3

=[1,3,5,7,5,-5,-8,-0.3,3,0,0,0]

 

4

=[2,4,6,8,0,0,-9,5,-4,0,3,-2]

 

5

=complex(A3,A4)

 

6

=A5.comimage()

7

=[1]

 

8

=A7.comimage()
