mae()

Description:

Calculate the mean absolute error between two vectors.

Syntax:

mae(A,B)

Note:

The function calculates the average absolute difference between vector A and vector B.

Parameters:

A/B

Vectors

Return value:

Numeric value

Example:

 

A

 

1

[3,5,2,8]

 

2

[4,6,2,4]

 

3

=mae(A1,A2)

1.5
