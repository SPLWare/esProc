tan()

Description:

Compute the tangent value.

Syntax:

tan(number)

Note:

The parameter number is in radians.

Parameter:

number

The radian value for which you want to compute the tangent value

Return value:

Float type

Example:

tan(pi()/2)

1.633123935319537E16

tan(pi(2))

-2.4492935982947064E-16

Related functions:

sin()

cos()
