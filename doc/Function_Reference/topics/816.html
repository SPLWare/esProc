ym_load_pcf()

Description:

Generate a model object based on the model building file.

Syntax:

ym_load_pcf(pdfile)

Note:

The external library function generates a model object according to model building file pdfile, which can be defined with an absolute path or a relative path. The default relative path is [sAppHome]/store/predict.

Parameters:

pdfile

A model building file

Return value:

A model object

Example:

 

A

 

1

=ym_env()

Start Python service

2

=ym_load_pcf("model.pcf")

Generate a model object based on the specified model building file

3

>ym_close(A1)

Release resources
