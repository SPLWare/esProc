A.mi()

Description:

Create indicator variable for missing values in a sequence variable.

Syntax:

A.mi()

Note:

The MathCli external library function (See External Library Guide) creates indicator variable for missing values in sequence variable A during modeling. If the missing value rate is 5% or 95%, do not generate the indicator variable and return null.

Parameter:

A

A sequence.

Return value:

Table sequence

Example:

 

A

 

1

[1,2,3,null,5]

 

2

=A1.mi()
