channel(ch)

Description:

Create a channel and push data in another channel into it.

Syntax:

channel(ch)

Note:

The function creates a channel, pushes data in channel ch into it, and returns a channel. This is equivalent to ch.push(channel()).

Parameter:

ch

A channel

Return value:

Channel
