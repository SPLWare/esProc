hdfs_download()

Description:

Download a local file to a specific position.

Syntax:

hdfs_download( hd, removeFile,localFile/localPath)

Note:

The external library function downloads the specified file or folder removeFile to the local path localPath, or as a specific file localFile.

Parameters:

hd

An hd connection

removeFile

To-be-downloaded file or folder

localFile

A local file

localPath

A local file name

Options:

@d

Download all files in the specified folder to the local machine, and require both the source and directories are folders

Return value:

Boolean

Example:

 

A

 

1

=hdfs_open(;"hdfs://192.168.0.8:9000")

Connect to HDFS file system

2

=hdfs_download(A1,"/emp/emp.txt","/test")

Download emp.txt in /emp folder to the local folder /test

3

=hdfs_download(A1,"/emp/emp.txt","/test/emp1.txt")

Download emp.txt in /emp folder to the local folder /test, and rename the file emp1.txt

4

=hdfs_download@d(A1,"/s1","/h1")

Download all files under folder /s1 to the local folder /h1

5

=hdfs_close(A1)
