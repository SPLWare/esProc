ifstring()

Description:

Find whether the specified data is string type.

Syntax:

ifstring(Exp)

Note:

The function finds whether the parameter Exp is of string type.

Parameter:

Exp

An expression of any data type

Return value:

Boolean value

Example:

ifstring("abc")

true

ifstring(1234)

false

ifstring("1980-01-01")

true

ifstring(date("1980-01-01"))

false
