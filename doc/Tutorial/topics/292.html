12.6.2 Appearance properties

As can be seen from the above chart, the end points and all other data points on the line are dots with automatically generated fill colors. A line element for plotting a polyline chart can be regarded as the dot elements connected with lines. In this sense, appearance properties of the line element are the same as those of the dot element discussed in The Dot Element. Note that in a polyline chart plotting, the lines of the polyline chart and the boundary line of the dots share properties such as line color, line weight and line style.
