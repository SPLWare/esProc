6.7.2   Starting non-GUI console

Use the non-GUI console under Linux, here is how to start it: execute the command ./ServerConsole.sh  -h (./ServerConsoleEE.sh  -h for the Enterprise Edition) in the esProc\bin directory under the installation root directory to start the HTTP service.
