12.4.2 Properties of axis title

You can set a title for an axis to render its data during plotting. The setting of title includes setting properties like font and style. Modify A4 in the plotting algorithm in the previous section into =A1.plot("EnumAxis","name":"x","axisColor":-16737895,"axisLineWeight":2,"axisArrow":256,"title":"Event","titleFont":"Times New Roman","titleStyle":7,"titleColor":-16776961) in which properties of the x-axis title are set as follows:
