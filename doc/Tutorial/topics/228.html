Chapter9      Heterogeneous Data Sources

In this big data era, the incredibly huge size of ever-increasing data isn’t the only challenge organizations are facing. Data is generated in and by more sources than the conventional databases, such as data files of various formats. This chapter explains how to handle data in those unconventional data soruces.
