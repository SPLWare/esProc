11.8.5             A brief introduction of DQL Server

DQL Server can be considered as a database that uses a pseudo table generated from a composite table or a bin fie as the data source, performs queries using DQL statements and returns result as the general JDBC result set.

After finishing the configuration file service.xml, configuring the metadata file (*.glmd) for DQL Server and doing related settings, we can start the DQL Server and connect to it through JDBC.

Find detailed explanations about DQL Server in esProc DQL Tutorial.
