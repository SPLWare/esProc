6.8.1   Downloading, installing and configuring JDBC Enterprise edition

Download esProc Enterprise edition installer in https://www.esproc.com/download-esproc/  and install it. After that, find the basic jars used by the Enterprise edition JDBC in [installation directory]\esProc\lib. Different from the other esProc editions, the jar file esproc-ent-xxxx.jar specifically used for the Enterprise edition is necessary. If you want to use the extended functionalities, add the functional jar esproc-ext-xxxx.jar.

Besides [installation directory]\esProc\config\raqsoftConfig.xml, files used to deploy the JDBC Enterprise can also be found in q-server\webapps\qvs\WEB-INF\home. The two configuration files have similar content, but the latter is used for server loading and will not adjust itself when the esProc edition changes. You need to edit it yourself as needed.

Read Scudata Cloud Deployment to find more about esProc Enterprise edition server.
