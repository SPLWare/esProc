3.5.3 Join operations

Another way of associating tables is SQL join operations. esProc join functions can achieve similar result. However, with the mechanism of referencing records and record sequences and the align function, join operations become less used.

Cross join

A cross join is a type of join operation that combines each member of the first sequence with each member of the second (or up to nth) sequence. It is the most basic join operation. Without any filtering condition, it simply lists all possible combinations. xjoin function is used to perform the operation. For example:
