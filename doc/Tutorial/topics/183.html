Chapter7      External Memory Computations

With the development of internet technologies, organizations are trying to cope with data with an ever-increasing size. Unlike the smaller conventional data tables, a massive amount data cannot be imported into the memory at a time. We need to batch import and process a huge amount of data with a cursor using the external memory computing. This chapter explains how to use the cursor to do that in esProc.
