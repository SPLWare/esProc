12.9.4 Barcode

In a text element, we can use barcode to present text. For example:

 

A

1

=canvas()

2

=A1.plot("Text","text":"12345678","barType":3,"dispText":true)

3

=A1.draw@p(200,200)

As shown below, you can set the barcode property, which is the last of the text properties:
