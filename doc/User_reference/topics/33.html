AutoSave

By checking this option, users can set the time interval for the autosave, which enables automatically storing all the currently open files at the backstage at every given time interval and auto-storing a file before it is closed or executed in case that the content is lost due to unexpected power blackout, mishandling, or any system errors.

It should work together with Auto-open Previous File. AutoSave turns gray when the latter is unselected.
