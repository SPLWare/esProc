Copy Value

Copy a cell value onto the clipboard for pasting.

The difference between the Copy and the Copy value is that the Copy is to copy the cell string.

Take =1+1 for example, a value 2 will be copied if using the Copy value, while a string =1+1 will be copied with the Copy action.
